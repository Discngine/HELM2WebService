HELM Web Editor

(Release 1, Jan 05, 2017)

HELM Web Editor is an editor to draw and display macro molecules, built with 100% javascript, depending on JSDraw.Lite, dojo toolkit and pako.js.

Download Binary

1. Download HELM Web Service for Tomcat

https://github.com/scilligence/HELM2WebService/blob/master/publish/WebService.warar

To deploy it on Tomcat, just copy the war file to Tomcat webapps folder (e.g. C:\Program Files\Apache Software Foundation\Tomcat 9.0\webapps)

2. For IIS web server, please contact Scilligence.
